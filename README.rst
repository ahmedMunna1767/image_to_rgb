================================
IMAGES to MIF file
===============================

1. Convert any Image to a pixel art. 
2. This module contains files to convert a image to its RGB pixel. 
3. The pixels are then written to a MIF file. MIF are Memory Initialization file. Which is used for loading internal ram/roms of Intel FPGA.
